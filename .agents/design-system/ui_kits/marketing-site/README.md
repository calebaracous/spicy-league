# Spicy League UI Kit

High-fidelity recreation of the Spicy League marketing site and core UI patterns. Built with React (Babel) for modular, reusable components.

## Structure

- `index.html` — Interactive homepage recreation with all major sections
- `README.md` — This file
- Component files (`.jsx`) loaded via Babel

## Usage

Open `index.html` to see the interactive homepage. Components are modular and can be copied into new designs.

All components follow the visual foundations from the parent design system: dark backgrounds, cream text, dark red accent, Geist Sans typography, and scroll-triggered reveal animations.
